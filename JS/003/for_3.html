- các loại vòng lặp trong js

        + vòng lặp for
            part 1 : sự khởi đầu của biến đếm
            part 2 : điều kiện chạy vòng lặp
            part 3 : sự thay đổi biến đếm trong vòng lặp

        cú pháp :
        for (part1; part2; part3) {
            // các câu lệnh thực thi bên trong vòng lặp
        }
